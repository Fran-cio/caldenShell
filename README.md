<!-- GETTING STARTED -->
## Getting Started
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Fran-cio/caldenShell.git
   ```
2. Make the project 
   ```sh
   make
   ```
3. Execute the generated bin `caldenShell`
   ```sh
   ./bin/caldel_shell
   ```
### Uninstallation

1. Use Make clean
    ```sh 
    make clean
    ```
